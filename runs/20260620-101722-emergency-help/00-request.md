# Request
쏘카 앱의 '사고·고장 신고(긴급 도움)' 화면을 디자인해줘. 이용 중 사고나 고장이 났을 때 빠르게 도움받는 화면이야.

구성:
- 상단: "도움이 필요하신가요?" 타이틀 + 안심시키는 한 줄.
- 긴급 액션(가장 강조): 긴급출동/고객센터 전화를 눈에 띄게. 상황의 위급함이 색으로 전달되게.
- 상황 선택: 사고 / 고장 / 기타 문의를 아이콘 리스트(또는 선택 박스)로.
- 주의 안내: "안전한 곳으로 이동한 뒤 신고해 주세요" 같은 Tips/Alert.
- 보조: 자주 묻는 질문, 현재 이용 중인 차량 정보.

상태: 기본 상태.
톤: 침착하고 신뢰감 있게. 급한 상황에서도 헷갈리지 않게, status 색을 적절히.

## Stated constraints / preferences
- 모바일 포트레이트, 한국어 카피, SOCAR 보이스(침착·신뢰감·명료).
- 정보 위계: 긴급 액션(긴급출동/고객센터 전화)이 가장 강조되어야 함.
- 위급함을 status 색으로 전달하되, 급한 상황에서 헷갈리지 않게 절제.
- 상황 선택: 사고 / 고장 / 기타 문의 (아이콘 리스트 또는 선택 박스).
- 주의 안내(Tips/Alert): "안전한 곳으로 이동한 뒤 신고해 주세요".
- 보조 정보: 자주 묻는 질문(FAQ), 현재 이용 중인 차량 정보.
- 상태: 기본 상태 1종.
