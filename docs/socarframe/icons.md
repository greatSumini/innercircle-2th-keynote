# Icons

The SOCAR Frame icon system — the SOCAR Icon Library, its naming convention, style variants, and the full set of available icon names.

Icons are a key visual element of the UI: they help users recognize and understand features quickly. SOCAR Frame provides an icon set covering a range of situations so the product keeps a consistent design language.

## Icon library

- **Library name:** SOCAR Icon Library
- **Total icons:** 230 (shown as `230/230` in the library)

## Naming convention

All icon names use the `icon-` prefix followed by a kebab-case descriptor:

```
icon-<name>[-<modifier>]-<style>
```

Most icons come in two **style variants**, encoded as the name suffix:

| Suffix | Style |
|---|---|
| `-fill` | Solid / filled icon |
| `-line` | Outlined / stroked icon |

Some icons exist in only one variant, and certain semantic or special icons (e.g. `icon-snackbar-*`, `icon-status-*`, `icon-background`) do not use the `-fill` / `-line` pairing.

## Categories

The library groups icons by purpose, including:

- **Directional / navigation** — arrows and chevrons (`icon-arrow-*`, `icon-chevron-*`, `icon-circle-arrow-right-*`, `icon-corner-up-right-fill`).
- **Letters & numbers** — circled and plain letter/number glyphs (`icon-a-fill`, `icon-b-fill`, `icon-c-circle-*`, `icon-one-circle-*` … `icon-nine-circle-*`, `icon-zero-circle-*`, `icon-26-up-fill`).
- **Mobility & transport** — `icon-car-*`, `icon-car-search-*`, `icon-bus-*`, `icon-subway-*`, `icon-train-*`, `icon-bus-*`, `icon-plane-departure-*`, `icon-luggage-*`, `icon-charging-*`, `icon-bolt-*`, `icon-gas-station-*`, `icon-droplet-*`, `icon-gauge-*`, `icon-horn-*`, `icon-transfer-*`, `icon-driver-license-*`.
- **Location & maps** — `icon-location-*`, `icon-location-dot-*`, `icon-location-like-*`, `icon-location-shape-*`, `icon-locationpill-line`, `icon-locationpin-fill`, `icon-map-line`, `icon-my-location-line`, `icon-spot-fill`.
- **Communication** — `icon-bell-*`, `icon-phone-*`, `icon-message-dots-*`, `icon-message-question-*`, `icon-mobile-line`, `icon-dialpad-fill`.
- **Status & feedback** — `icon-check-*`, `icon-check-circle-*`, `icon-exclamation-circle-*`, `icon-exclamation-triangle-*`, `icon-info-circle-*`, `icon-question-circle-*`, `icon-warning-*`, `icon-snackbar-caution`, `icon-snackbar-success`, `icon-snackbar-warning`, `icon-status-caution`, `icon-status-success`, `icon-status-warning`.
- **Actions & controls** — `icon-plus-*`, `icon-minus-*`, `icon-close-*`, `icon-edit-*`, `icon-search-*`, `icon-filter-*`, `icon-setting-*`, `icon-refresh-*`, `icon-reload-*`, `icon-share-line`, `icon-copy-line`, `icon-download-line`, `icon-zoom-in-line`, `icon-zoom-out-line`, `icon-trash-*`, `icon-menu-line`, `icon-dots-line`, `icon-radio-on-line`, `icon-radio-off-line`.
- **Commerce & documents** — `icon-coupon-*`, `icon-couponset-*`, `icon-credit-card-*`, `icon-coin-*`, `icon-file-check-*`, `icon-file-text-*`, `icon-cardscan-line`, `icon-briefcase-*`.
- **People & profile** — `icon-user-*`, `icon-users-*`, `icon-profile-*`, `icon-smile-circle-*`.
- **General UI / objects** — `icon-home-*`, `icon-heart-*`, `icon-bookmark-*`, `icon-clock-*`, `icon-calendar-check-*`, `icon-camera-*`, `icon-photo-*`, `icon-photoset-*`, `icon-video-*`, `icon-eye-*`, `icon-lock-*`, `icon-bluetooth-*`, `icon-flashlight-*`, `icon-flame-*`, `icon-thumb-up-*`, `icon-thumb-down-*`, `icon-buildings-*`, `icon-background-circle-fill`, `icon-background-triangle-fill`, `icon-background`.

## Usage

Use icons to help users recognize and understand features quickly, and keep a consistent design language by drawing only from the SOCAR Icon Library. Choose the `-fill` (solid) or `-line` (outline) variant to match the emphasis and visual weight needed in context.
